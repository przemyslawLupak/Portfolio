Pokedex

front-end

Work Description

Language: JavaScript/Html/CSS/TypeScript


Tools: Angular/Bootstrap


Date: 04.04.2020-16.04.2020


Pokedex is the first project in which I used foreign API, made in Angular and using pokeapi.co. API was not a problem for me after reading the documentation, but I spent a lot of time for polishing it. For the first time I had to use data catching. In my opinion, the final effect is adequate and befitting, and I'm satisfies with it in terms of functionality. I think the only aspect I should spend more time to work on with is User Interface. Design is based on Bootstrap and is very modest, but as a result of displaying the application on the smaller screens such as phones is a success. For the time devoted to this project is nearly two weeks due to the deadline, but I’m going to refresh the interface by using CSS more in near future.
